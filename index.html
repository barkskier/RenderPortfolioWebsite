Spreadsheet Applicaton

An advanced spreadsheet application resembling Google Sheets and Microsoft Excel. Empowers users with dynamic cell dependencies through powerful formulas and offers the flexibility to effortlessly save and share spreadsheet files across various devices.

Sprite Editor

Drawing inspiration from Piskel and other leading sprite editors, this application provides users with a rich array of features, including adjustable brush size, fill brush, vibrant color palettes, customizable animation speed, and more. Users can save their current frames to a personal device and seamlessly share their creative work with others.

Agario

Derived from the popular game AGAR.io, this application mirrors its functionality, enabling multiple players to congregate on a shared server for collaborative gameplay. Offering an array of features such as a dynamic leaderboard, real-time tracking of the highest score in each round, and individualized personal best records, our platform delivers an engaging and interactive gaming experience.

Educational Game

An educational application designed to enlighten users about the consequenses of impaired driving. Drawing inspiration from the popular mobile game 'Smashy Road,' this application aligns itself with the mission of addressing the hazards associated with impaired driving. By making impactful decisions within the simulated environment, users encounter informative content highlighting both responsible and risky behaviors before taking control of the wheel.
